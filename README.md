# Svete Scorm Packager

This project is an attempt to make a scorm template for svelte. It is aimed to provide a great developer experience for developers who also write courses, and they are alwasy limited by the existing tools.

During the making of this project many other packages and utilities could be made. I will try to make those tools input separate packages if they can serve a wider audience.

## Developer

To run the project, you need to have node and npm installed. Then you can run the following commands:

```bash
pnpm install
pnpm turbo run dev
```
